# 皖美的她 — 绒花非遗手作 H5

一个纯前端、无需服务器的非遗绒花手工作品展示与管理平台，内置可视化后台，支持手机端浏览。

## 功能

### 前端展示
- 搜索与分类筛选（耳饰/戒指/发卡/胸针/发簪/装饰作品/其他）
- 两列网格卡片布局，点击查看详情大图
- 关于页面（品牌故事、数据卡片、联系方式）
- 团建定制页面（套餐展示 + 往期活动图片）

### 管理后台（密码：admin123）
- 作品增删改查 + 图片上传（Canvas 自动压缩）
- 团建套餐卡片编辑
- 团建图片批量上传与管理
- 数据导出/导入 JSON

## 使用方式

直接双击 `index.html` 在浏览器中打开即可，无需安装任何依赖。

## 技术栈

- 纯原生 HTML / CSS / JavaScript
- 数据存储在浏览器 localStorage
- Canvas 图片压缩
